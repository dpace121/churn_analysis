# churn_analysis
📊 Project Overview
This project analyzes a telecom dataset to uncover the drivers behind a 26.54% churn rate. Key focus areas include contract types, payment methods, and service tiers (Fiber Optic).

🚀 Key Findings
Contract Risk: Month-to-month contracts are the strongest predictor of churn.
Payment Friction: Customers using Electronic Checks churn at the highest rates.
Service Gaps: High-tier Fiber Optic users show significant price sensitivity/dissatisfaction.
Demographics: Senior citizens exhibit unique usage patterns requiring tailored retention.

🛠️ Tech Stack
Language: Python
Libraries:
Pandas: Data cleaning and transformation
Matplotlib & Seaborn: Statistical data visualization

📁 Analysis Workflow
Data Cleaning: Handled missing TotalCharges, formatted data types, and verified unique records.
EDA: Visualized demographic impacts and service correlations using count plots and distributions.
